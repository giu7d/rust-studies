# Rust Studies

## Getting started

### Prerequisites

- Cargo (1.41.0);
- Rustc (1.41.0).

### Installing

```
git clone <repo>
cd <repo>
```

### Running

```
cd <repo>
cargo run
```

### Deploying

#### Development

```
cd <repo>
cargo build
```

#### Production

```
cd <repo>
cargo build --release
```
